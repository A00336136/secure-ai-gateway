AI Safety Guardrails — 3-Layer Defence-in-Depth

NeMo Guardrails (Colang DSL) → LlamaGuard (MLCommons S1-S12) → Presidio (PII).

  • Layer 1 (NeMo): Blocks jailbreak attempts with Colang rules
  • Layer 2 (LlamaGuard): Classifies harmful content against S1-S12 taxonomy
  • Layer 3 (Presidio): Detects PII entities (email, SSN, credit card)
