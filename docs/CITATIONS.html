How to Cite This Project

If you use or reference the SecureAI Gateway in your research, please use one of the following citation formats:

BibTeX

IEEE

A. Shaik, "SecureAI Gateway: A Defence-in-Depth Architecture for Securing LLM Applications with Multi-Layer Guardrails and DevSecOps Integration," 2024. [Online]. Available: https://github.com/afrozshaik/secure-ai-gateway

APA

Shaik, A. (2024). SecureAI Gateway: A defence-in-depth architecture for securing LLM applications with multi-layer guardrails and DevSecOps integration. https://github.com/afrozshaik/secure-ai-gateway
